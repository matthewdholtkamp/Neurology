# Headache — overview & triage

<span class="reviewed-stamp">Last reviewed: July 2026</span>

The first job in any headache patient is not to diagnose migraine — it is to **exclude a
secondary headache** that will hurt the patient if you miss it. Triage on red flags first,
then treat the primary headache.

!!! danger "🚩 Red flags — stop and image / work up"
    Use the **SNNOOP10**-style screen. Any of these changes your pathway:

    | Pattern | Worry | Do now |
    |---|---|---|
    | **Thunderclap** (max intensity < 1 min) | SAH, RCVS, dissection, CVST | **Non-contrast CTH now**; if negative and suspicion remains, LP (xanthochromia) ± CTA/MRV |
    | Systemic signs — **fever, stiff neck, rash, immunosuppression/HIV/cancer** | Meningitis, abscess, mets | Labs, **LP** (imaging first if focal deficit/↓LOC/papilledema), empiric coverage |
    | **Focal deficit, seizure, altered mental status, papilledema** | Mass, stroke, ↑ICP | **MRI brain** (CT if emergent); if papilledema add **MRV** |
    | **Positional** — worse standing (or worse lying) | Intracranial hypotension / hypertension | MRI brain ± spine; consider LP with opening pressure |
    | **Precipitated by Valsalva / exertion / sex** | SAH, posterior fossa lesion, RCVS | CTH + **CTA head/neck**; MRI outpatient |
    | **New or progressive after age 50** | GCA, mass | ESR/CRP (GCA), MRI |
    | **New headache in pregnancy/postpartum** | CVST, PRES, pre-eclampsia, pituitary | MRI/MRV, BP, labs |
    | **Change in an established pattern** | New secondary process | Re-work up as new headache |

    *Reassuring:* an established, stereotyped, self-resolving headache with a **normal exam**
    and **no red flags** — that is where primary headache lives.

## Rapid approach

1. **Screen for red flags** (table above). Any hit → secondary work-up pathway.
2. **Normal exam + no red flags?** Classify the primary headache by phenotype.
3. **Route to the diagnosis:**
    - Recurrent, moderate–severe, disabling, ± photophobia/phonophobia/nausea/aura →
      **[Migraine](migraine.md)** *(if it reached neurology, it is almost never "just tension")*
    - Severe **unilateral** attacks with **ipsilateral autonomic features** (tearing,
      ptosis, rhinorrhea), restless/agitated → **[Trigeminal autonomic cephalalgias](cluster-tac.md)**
      (cluster, paroxysmal hemicrania, SUNCT/SUNA)
    - Brief, electric, **triggered** stabs in a trigeminal division → **[Trigeminal
      neuralgia](trigeminal-neuralgia.md)** — and remember **MS can present this way**
    - Headache **≥15 days/month with acute-med overuse** → **[Medication overuse
      headache](medication-overuse.md)**
    - **Daily and unremitting from a remembered onset**, no prior headache history →
      **[New daily persistent headache](new-daily-persistent.md)**
    - Headache after head/neck trauma → **[Post-traumatic headache](post-traumatic.md)**

## Exam essentials

Vitals (including **BP** and temperature), **fundoscopy for papilledema**, cranial nerves,
visual fields, focal motor/sensory screen, gait, neck stiffness, and **temporal artery**
palpation in older patients. A normal neuro exam is a large part of what makes a primary
headache diagnosis safe.
